## 简介

目前，Java 在国内的使用比重越来越重，并且它也是一门学习曲线相对来说比较陡峭的语言；针对于许多小伙伴不知道该怎么来深入学习 Java 的情况，决定发起该项目，为国内整个 Java 社区整理一份高质量、范围广的 Java 面试指南；旨在帮助广大的Java 开发者指引方向，提升技术，找到心仪的工作。
